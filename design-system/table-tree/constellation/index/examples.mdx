## Basic

Table tree displays a table with expandable nested rows. The basic table tree has configurable props for header titles,
columns, column widths, and an array of table items (with child items).

The `content` of each item is customizable, but your data must otherwise conform to the expected schema.

<Example Component={BasicTableTreeExample} packageName="@atlaskit/table-tree" />

## Advanced

For more advanced usage, you can compose `Cell`, `Header`, `Headers`, `Row`,
and `Rows` components.

This approach allows you to structure your data however you wish.

## Uncontrolled

The expansion and collapsion of each row is handled automatically.

Use the `isDefaultExpanded` prop to change whether a row is expanded by default.

  Component={UncontrolledTableTreeExample}

## Controlled

Use the `isExpanded` prop to manually control whether a row is expanded or not.

Respond to user interaction by providing callbacks to the `onExpand` and
`onCollapse` props.

## Loading states

Each nesting level can be displayed in a loading state. This is useful when
loading data asynchronously.

### Root

To display the root level in a loading state, ensure that the `items` value
provided to `Rows` is `undefined`.

<Example
  Component={LoadingStateRootTableTreeExample}
  packageName="@atlaskit/table-tree"
/>

### Nested

To display a nested level in a loading state, ensure that the `items` value
provided to the corresponding `Row` is `undefined`.

<Example
  Component={LoadingStateNestedTableTreeExample}
  packageName="@atlaskit/table-tree"
/>

## Expand and collapse on row click

Use the `shouldExpandOnClick` prop to change whether a row with children expands when clicked anywhere on a row instead of only via the chevron.

Note that **this should not be used when interactive elements like a button or dropdown menu exist within your row**. Clicking on those elements will trigger the row's expanding or collapsing along with the default behavior of the interactive element.

  Component={ShouldExpandOnClickTableTreeExample}

## Extended aria label for expand and collapse button

Use the `mainColumnForExpandCollapseLabel` prop to add detail to the expand and collapse button visually hidden text readout.
The prop accepts a column's component name and adds the text from that column's cell to the row's expand button.

For example, the first expand button below would read "Expand Chapter 1: Clean Code row".
