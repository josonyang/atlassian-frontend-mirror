## ensure-design-token-usage

Ensures that the codebase uses the global `token` function, rather than hard-coded values. This ruleset is great for codebases that are starting to adopt tokens and those that have already adopted tokens. This ruleset also prevents new contributors from accidentally adding hard-coded color or spacing values.

üëé Example of **incorrect** code for this rule:

```css
color: red;
       ^^^
```

```css
box-shadow: 0px 1px 1px #161A1D32;
                        ^^^^^^^^^
```

```css
padding: 8px 16px;
         ^^^^^^^^
```

```css
color: var(--not-a-token-value);
       ^^^^^^^^^^^^^^^^^^^^^^^^
```

üëç Example of **correct** code for this rule:

```css
box-shadow: var(--ds-shadow-raised);
```

```css
color: var(--ds-text-highEmphasis);
```

```css
padding: var(--ds-space-100) var(--ds-space-200);
```
