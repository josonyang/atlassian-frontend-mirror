## Default

The default icon button, used for most cases. They aren't impactful enough to represent a primary action.

<Example Component={IconButtonDefault} packageName="@atlaskit/button" />

## Tooltips

<SectionMessage appearance="information">
  Tooltips are currently disabled by default. They will be enabled by default in
  a later release. Prefer using built-in tooltips over wrapping the icon button
  manually.
</SectionMessage>

Set `isTooltipDisabled` to false to enable the tooltip. The value of `label` will be used for the tooltip content.

<Example Component={IconButtonTooltip} packageName="@atlaskit/button" />

### Overriding tooltip props

Use the `tooltip` prop to override the default tooltip props. It accepts all [tooptip props](/components/tooltip/code), excluding `children`.

<Example Component={IconButtonTooltipOptions} packageName="@atlaskit/button" />

## Appearance

### Primary

A primary button calls attention to the most important action on a page or in an area. Primary icon buttons aren't common because critical actions should typically use [a button with a label for clarity](/components/button/button-new).

<Example Component={IconButtonPrimary} packageName="@atlaskit/button" />

### Subtle

Use a `subtle` icon button for secondary actions.

<Example Component={IconButtonSubtle} packageName="@atlaskit/button" />

## Spacing

Icon button spacing depends on the surrounding UI. Default spacing is used for most use cases and `compact` for tables or smaller spaces.

## Shape

### Circle

Only use circle icon buttons in the top navigation or other areas that already have circular buttons.

## States

### Disabled

Set `isDisabled` to disable an icon button that shouldn't be actionable. The icon button will appear faded and won't respond to user interaction.

Avoid disabling buttons because this can cause accessibility problems, and [never put a tooltip on a disabled button](/components/button-new/usage#do-not-put-tooltips-on-disabled-buttons).

<Example Component={IconButtonDisabled} packageName="@atlaskit/button" />

### Selected

Set `isSelected` to indicate the button is selected.

<Example Component={IconButtonSelected} packageName="@atlaskit/button" />

### Loading

When a button is still loading and not actionable, a loading spinner can be displayed in place of the icon. This also disables the button and blocks user interaction.

<Example Component={IconButtonLoading} packageName="@atlaskit/button" />
