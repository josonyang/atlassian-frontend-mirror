import SelectRadio from '../../examples/constellation/select-radio';

Use the radio select component for dropdown options displayed with radio buttons.

Radio buttons can be used for a list of options where only one choice can be selected. If you need
to have multiple selectable options, use [multi select](/components/select/examples#multi-select) or
[checkbox select](/components/select/checkbox-select) instead.
