## Small (12px)

<SectionMessage appearance="discovery" title="New" icon={PremiumIcon}>
	Small icons are now available, providing a fully downscaled set of icons.{' '}
	<strong>These replace utility icons</strong>, which are deprecated.

## Utility (12px)

<SectionMessage appearance="warning" title="Deprecated">
	Utility icons are deprecated in favor of core icons, which will now support a new{' '}
	<inlineCode>size</inlineCode> prop set to <inlineCode>small</inlineCode>. This change aligns with
	design requirements for a greater variety of smaller-sized icons. Consequently,{' '}
	<strong>all icons will now be available in this smaller size</strong> instead of a limited
	selection.
</SectionMessage>

Utility icons are 12px for special cases and component parts that require a smaller size. Some
glyphs aren't legible at this size, so only certain icons are offered at 12px.

<Example Component={IconUtility} packageName="@atlaskit/icon" hiddenSections={['iconography']} />

## Label

If an icon doesn’t have an existing text label or accessible text, provide a clear label with the
`label` prop.

If an icon is associated with a button or element that also has a text label, you don’t need to
provide alternative text for the icon, because the label clarifies the meaning of the icon. You can
do this by setting the `label` prop to an empty string (`""`).

<Example Component={IconLabel} packageName="@atlaskit/icon" hiddenSections={['iconography']} />

## Color

Icons can use color tokens for icons, text, links, or the current text color.

<Example Component={IconColor} packageName="@atlaskit/icon" hiddenSections={['iconography']} />

## Spacing props

Icons support a `spacing` prop to add padding around the icon. `spacious` spacing sets the bounding
box to `24px`. Both small and utility icons support an additional option, `compact`, that sets the
bounding box to `16px`.

<Example Component={IconSpacing} packageName="@atlaskit/icon" hiddenSections={['iconography']} />

## Migration

Atlassian apps can switch between the old and new icon components using feature flags and the
`migration` entrypoint for the new icons, or switch manually using the `LEGACY_fallbackIcon` prop.

<Example Component={IconMigration} packageName="@atlaskit/icon" />

To learn more about feature flagging and `LEGACY_` prefixed props, see the
[icon migration guide](/components/icon/migration-guide#atlassian-developers).
