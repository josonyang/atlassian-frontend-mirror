Icon tile places an icon on a background of a specified size, shape and color. It allows icons to be
prominently highlighted in a layout, such as next to a heading.

## Size

Unlike standard icons, icons inside Icon Tiles can scale up and down for use in different layouts.

<SectionMessage title="Size changes" appearance="information">
	<p>Changes are coming to Icon Tile sizing:</p>
	<ul>
		<li>Pixel number values are being replaced by t-shirt sizes.</li>
		<li>
			Size <inlineCode>"16"</inlineCode> is deprecated and should be replaced with an alternative,
			such as direct <a href="/components/icon/examples">Icons</a> without a tile or enlarging to
			the next available size <inlineCode>"xsmall"</inlineCode>.
		</li>
		<li>
			The default size will change from <inlineCode>"24"</inlineCode> to{' '}
			<inlineCode>"medium"</inlineCode> (32px).
		</li>
	</ul>
	<p>These changes will occur in upcoming releases, and both APIs will be supported until then.</p>
</SectionMessage>

All available sizes:

- `16` – <Lozenge appearance="moved" isBold>Deprecated</Lozenge>
- `xsmall` – <Lozenge appearance="new" isBold>New</Lozenge> (20px)
- `small`, equivalent to `24`
- `medium`, equivalent to `32`
- `large`, equivalent to `40`
- `xlarge`, equivalent to `48`

## Shape

Icon Tile is available in two shapes: `square` and `circle`.

<SectionMessage title="Deprecated" appearance="warning">
	<p>
		The <inlineCode>shape</inlineCode> prop is deprecated and will be removed in a future version,
		meaning all Icon tiles will only be available in a square shape rather than a circle. Consider
		migrating to alternatives such as a square tile, an{' '}
		<a href="/components/button/icon-button/examples">Icon Button</a> for interactive elements, or
		rebuilding the component custom using ADS primitives.
	</p>
	<p>
		Since the circle shape is deprecated, Icon Tiles with this shape will not be affected by the
		feature flag that updates tile to the new design language.
	</p>
</SectionMessage>
