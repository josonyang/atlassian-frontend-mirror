## Default

## Migration

Atlassian apps can switch between the old and new icon components using feature flags and the
`migration` entrypoints for the new icons, or switch manually via the `LEGACY_fallbackIcon` prop.

<Example Component={IconMigration} packageName="@atlaskit/icon" />

To learn more about feature flagging and `LEGACY_` prefixed props, see the
[icon migration guide](/components/icon/migration-guide#atlassian-developers).

## Sizing

### Small

The `small` icon size (16px) is used when space is at a premium, such as search results or inline
content.

<Example Component={IconSmall} packageName="@atlaskit/icon" hiddenSections={['iconography']} />

### Medium

The `medium` icon size (24px) is used in common interfaces.

<Example Component={IconMedium} packageName="@atlaskit/icon" hiddenSections={['iconography']} />

### Large

The `large` icon size (32px) is used in common interfaces.

<Example Component={IconLarge} packageName="@atlaskit/icon" hiddenSections={['iconography']} />
