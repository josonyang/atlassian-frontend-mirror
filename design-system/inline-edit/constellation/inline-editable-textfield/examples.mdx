## Default

The default form of the inline editable text field allows for easy switching between read-only text and editable text on the same page.

Most inline edit use cases use a standard text field. This component includes our recommended defaults for this use case. Use an [inline edit](/components/inline-edit) if you require more customisation and need to pass in your own custom input components.

<Example
  Component={inlineEditableTextfieldDefault}
  packageName="@atlaskit/inline-edit"
/>

## Compact

The height of the inline edit can be decreased through `isCompact`. The top and bottom padding decreases.
