These components are designed to go in the
[side nav layout area](/components/navigation-system/layout/examples#side-nav).

## Menu items

<SectionMessage>
	Menu item components from this package should only be used as direct children of the side nav. If
	you are composing your own custom menus, for example with popups or dropdown menus, please see the{' '}
	<a href="#composing-menus">composing menus</a> section.
</SectionMessage>

### Link menu item

The link item component renders a menu item wrapped in an anchor tag `<a>`. This is the most common
type of menu item, as most menu items are used to send people to another location.

For menu items that do something else, use the button item component instead.

Use the [app provider](/components/app-provider/examples) to specify a custom router link component.

<Example Component={LinkMenuItemExample} />

### Button menu item

The button item component renders a menu item wrapped in a button tag `<button>`. Use this component
when you have an action that does something other than navigating to a new page or context.

#### Disabled state

When disabled, content in the `actions` and `actionsOnHover` props will not be rendered.

<Example Component={ButtonMenuItemDisabledExample} />

### Expandable menu item

The expandable item component allows you to nest menu items. Use this component to form navigation
hierarchies.

<Example Component={ExpandableMenuItemExample} />

### Flyout menu item

The flyout menu item renders a popup with the menu item as the trigger. Use this component to render
children that aren't menu items, such as search bars and filters.

You can also render additional menu items within the flyout. For these, you should use the other
menu items from the `@atlaskit/navigation-system` package.

<Example Component={FlyoutMenuItemExample} />

### Composing menus

You can use the `actions` or `actionsOnHover` props to compose your own menus. If you want to render
additional menu items within these menus, you should use either:

- menu item components from [@atlaskit/dropdown-menu](/components/dropdown-menu/examples)
- a combination of [@atlaskit/popup](/components/popup/examples) and
  [@atlaskit/menu](/components/menu/examples).

You shouldn't use `@atlaskit/navigation-system` menu items within these menus.

<Example Component={ComposingMenus} />

## Sections

Use `MenuSection` to group related items. Screen readers will convey these groupings.

Optionally compose with `MenuSectionHeading` to label the group, and `Divider` to render a
separator.

<Example Component={MenuSectionExample} />

### Spacing

Use `TopLevelSpacer` to create spacing between:

- global app shortcut section
- 'more' button menu item
- starred spaces (in Confluence)
