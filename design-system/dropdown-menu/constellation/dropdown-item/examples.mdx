## Multiline

Use `shouldTitleWrap` to let the dropdown item's visible label wrap over multiple lines.
The default is `true`. We recommend content wrapping over truncation, because truncation isn't usable or accessible.

<Example
  Component={DropdownItemMultiline}
  packageName="@atlaskit/dropdown-menu"
/>

Use `shouldDescriptionWrap` to let the description wrap over multiple lines.
Defaults to `true`. We recommend content wrapping over truncation, because truncation isn't usable or accessible.

<Example
  Component={DropdownItemDescriptionMultiline}
  packageName="@atlaskit/dropdown-menu"
/>

## States

### Disabled

Use `isDisabled` to make the element appear disabled.
This will remove interactivity and hide it from people who use assistive technology, so use this state with caution.

<Example
  Component={DropdownItemDisabled}
  packageName="@atlaskit/dropdown-menu"
/>
